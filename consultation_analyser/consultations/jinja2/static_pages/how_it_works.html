How it works

How the AI finds the themes and summarises them:

  • Responses to the consultation are organised by similar themes for each question.
  • These themes generate a list of consultation themes.
  • Individual lists of themes are combined into one list for the entire consultation.
  • The LLM checks each response to see which theme it contains.
  • The LLM generates a summary of the responsesâ€™ themes.

You can see the code on https://github.com/i-dot-ai/consultation-analyser
