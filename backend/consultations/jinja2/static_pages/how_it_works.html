How it works

How AI finds themes in the consultation and summarises them:

  • For each question, free-text responses are clustered into themes using a machine learning technique called topic modelling
  • For each theme, a large language model (LLM) is used to create a human-readable summary
  • The tool uses an internally hosted LLM suitable for Official-Sensitive data

You can see the code on https://github.com/i-dot-ai/consultation-analyser
